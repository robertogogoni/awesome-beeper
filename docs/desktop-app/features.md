# Beeper Desktop Features

> Last updated: 2026-01-21

## Core Features

### Unified Inbox
`[official]` All your chats from 15+ networks in one place.

### On-Device Bridges
`[official]` Privacy-preserving bridges that run locally.

- WhatsApp, Instagram, Messenger
- Telegram, Signal, Discord
- And more...

### Desktop API
`[official]` REST API for automation and integrations.

- Enable in: Settings → Developers → Beeper Desktop API
- Default: `http://localhost:23373`
- [SDK Documentation](../sdk/changelog.md)
